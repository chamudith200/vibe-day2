This repository contains two simple Python scripts developed as part of the Vibe Day 02 exercises.

## 1. Grade Calculator (`grade_calculator.py`)
A script that calculates student grades based on marks from three subjects.

### Features:
- Asks for the student's name and marks for three subjects.
- Calculates the average mark.
- Assigns a grade based on the following criteria:
  - **Grade A**: Average ≥ 75
  - **Grade B**: 60–74
  - **Grade C**: 40–59
  - **Fail**: Below 40
- Runs repeatedly until the user types `Exit`.
- Displays clean, formatted output.

### How to Run:
```bash
python grade_calculator.py
```

## 2. Monthly Budget Tracker (`monthly_budget.py`)
A script to track monthly expenses against a total budget.

### Features:
- Asks for the total monthly budget.
- Allows multiple expense entries until the user types `done`.
- Subtracts total expenses from the budget to show the remaining balance.
- **Low Funds Warning**: Displays a "Warning: Low Funds" message if the remaining balance is less than 500 LKR.

### How to Run:
```bash
python monthly_budget.py
```
